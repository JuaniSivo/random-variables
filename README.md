# Probability Distribution Generator

A Streamlit app to generate, visualize, and save probability distributions.

## Features
- Uniform, Gaussian, Lognormal, and Triangular distributions
- Optional truncation for Gaussian and Lognormal
- Histogram and CDF plots
- Save distributions to a table
- JSON-based storage

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
